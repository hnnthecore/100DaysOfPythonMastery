### ğŸ’¡ Project Overview
The **AI Personal Knowledge Engine** is a lightweight, local AI-style system that stores knowledge and answers user questions through **semantic reasoning**, not hardcoded responses.

Instead of relying on machine learning libraries, this project demonstrates how **early expert systems and modern AI agents work internally**:
- Information is stored as structured knowledge  
- User questions are analyzed for intent  
- Relevant knowledge is scored and ranked  
- The best answer is returned with an explanation  

This project focuses on **reasoning, transparency, and system design**, not black-box AI.
