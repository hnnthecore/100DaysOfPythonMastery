## Overview
**SoundWave** is a clean, lightweight desktop **Audio Player GUI** built with Python.  
It allows users to load, play, pause, and stop MP3/WAV files through an intuitive interface.

This project focuses on:
- GUI development with Tkinter  
- Audio playback handling  
- Event-driven programming  
- Clean project structure without unnecessary complexity  

No web scraping.  
No automation nonsense.  
Just a solid, practical desktop app.

---

## Features
- Load multiple audio files at once  
- Supports **MP3** and **WAV** formats  
- Play, Pause, Resume, and Stop controls  
- Playlist display with song selection  
- Displays currently playing track  
- Stable and beginner-friendly architecture  

---

## Tech Stack
- **Python 3.10+**
- **Tkinter** (GUI)
- **Pygame** (Audio engine)
