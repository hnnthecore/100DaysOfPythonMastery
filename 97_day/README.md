### üí° Project Overview
The **Smart Decision Advisor** is a lightweight, interactive Python program that helps users make decisions based on their personal priorities.

Instead of hardcoded outcomes, the system:
- Collects user preferences through guided questions  
- Applies weighted rules to different options  
- Scores each possible path  
- Recommends the most suitable choice  
- Explains *why* that choice was selected  

This project focuses on **logic, reasoning, and clarity**, not UI or backend complexity.
